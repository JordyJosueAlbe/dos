Resumen

El instructor del curso le hace una pagina web a un restaurante de comida china, aconseja que cuando un programador haga una pagina para una empresa les muestre otras paginas para que ellos digan que estilos les gusta mas y para asi hacer una parecida, tambien aconseja que hay que ofrecer un numero de revisiones limitado por ejemplo 3 para que asi si el cliente quiere mas revisiones le tiene que pagar mas al programador. Yeikov utiliza plantillas de boostrap para ponerle iconos a su pagina el resto de programacion y diseño lo hace con css. La estructura con html y la configuracion de la pagina con javascript.
